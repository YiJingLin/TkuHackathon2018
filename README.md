# GoeSo
![logo](./public/assets/img/logo.png)

## Idea
1. 透過相互幫助或發起活動，建立或加強人與人之間的連結。
2. 網路的連結最終還是得回到現實生活。

## Features
1. 遊戲化(**Gamification**)
2. 串連地理資訊(**Google Map API**)，找出附近的用戶建立連結
3. 參與或完成活動可獲得點數
4. 透過獲得的點數，強化自己的角色，或解鎖系統中的功能。

## Technique
1. Web App (front-end)
2. framework : bootstrap v3.3
3. google map API
